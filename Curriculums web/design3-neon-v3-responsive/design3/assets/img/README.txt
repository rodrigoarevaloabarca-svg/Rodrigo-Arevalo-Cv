=== CARPETA IMÁGENES ===

1. FAVICON (favicon.svg)
   - Ya incluido. Se muestra en la pestaña del navegador como "RA" con gradiente neon.
   - Puedes reemplazarlo con uno personalizado manteniendo el mismo nombre.

2. FOTO DE PERFIL (profile.jpg)
   - Coloca tu foto en esta carpeta con el nombre exacto: profile.jpg
   - Tamaño recomendado: 400x400px o más (cuadrada)
   - Formatos aceptados: .jpg, .jpeg, .png, .webp
   - Si cambias el nombre, actualiza src="assets/img/profile.jpg" en index.html
   - Si no hay foto, el sitio muestra automáticamente "RA" como initiales fallback.
